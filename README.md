### Requirements
* PHP 8.1
* Node ^14.0

<a name="step2"></a>
### Installation
* cp `.env.example` to `.env`
* configure `.env` for database credentials
* `composer install`
* `php artisan migrate`
* `npm install`

<a name="step3"></a>
### Test
* `php artisan test`

<a name="step4"></a>
### Run
* `php artisan serve`
* `npm run dev`
